Questions fréquentes

C'est quoi exactement comme technologie ?

Je construis une application web avec Python — une vraie interface que vous ouvrez dans votre navigateur, comme un site web. Vos données restent dans Google Sheet ou dans une base de données sécurisée selon vos besoins. Pas de logiciel à installer pour vos utilisateurs.

Comment est hébergée l'app et qui y a accès ?

L'application est hébergée sur un serveur sécurisé (cloud ou votre propre infrastructure selon vos préférences). L'accès est protégé par identifiant et mot de passe — seules les personnes que vous autorisez peuvent se connecter. Je m'occupe de toute la configuration.

Est-ce que mes données restent dans Google Sheet ?

Oui si vous le souhaitez. L'app peut se connecter à votre Sheet existante — vos données ne bougent pas. On peut aussi migrer vers une base de données plus robuste si votre volume de données le justifie.

Faut-il des compétences techniques pour utiliser l'app ?

Aucune. Je vous forme en 1h et l'interface est pensée pour être intuitive. La plupart des clients sont autonomes dès le lendemain de la livraison.

Et si je veux modifier l'app plus tard ?

Les modifications sont incluses pendant 30 jours après livraison. Pour les évolutions ultérieures, je propose un tarif horaire à 80€/h ou un forfait maintenance mensuel.
